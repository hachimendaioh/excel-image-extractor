# 📊 Excel 画像抽出ツール

ExcelファイルにEmbeddedされた画像を抽出し、ファイル名を編集してダウンロードできるブラウザツール。インストール不要・サーバー不要。

## 🔗 使ってみる

**[► ツールを開く](https://hachimendaioh.github.io/excel-image-extractor/)**

---

## ✅ 機能

- `.xls` / `.xlsx` / `.xlsm` すべてに対応
- **PNG・JPEG・GIF** を自動検出・抽出
- 各画像のファイル名をその場でテキスト入力で編集
- **プレフィックス**（接頭辞）を全画像に一括付与（例: `魚久2026_`）
- 個別ダウンロード / ZIP一括ダウンロード
- ファイル名をクリップボードにコピー

---

## 🔒 プライバシー・セキュリティ

すべての処理はブラウザ内で完結。  
**ファイルデータは外部サーバーに一切送信されません。**  
社外秘・機密情報が含まれるExcelファイルにも安全に使用できます。

## 🛠 技術仕様

| 項目 | 内容 |
|------|------|
| 言語 | HTML5 / CSS3 / Vanilla JavaScript |
| 外部ライブラリ | JSZip 3.10.1（ZIP保存のみ） |
| xls対応 | 自前OLE2パーサー + BIFF8 Escherストリーム復元 |
| xlsx対応 | ZIPを展開し `xl/media/` から直接取得 |

---

## 📝 使い方

1. 上のリンクからツールを開く
2. Excelファイルをドロップ（または選択）
3. 画像が自動で抽出・表示される
4. `PREFIX` 欄に共通の接頭辞を入力（任意）
5. 各画像のファイル名を編集（任意）
6. 「↓ 保存」または「↓ すべて保存（ZIP）」でダウンロード
