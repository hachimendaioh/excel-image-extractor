# Excel 画像抽出ツール

ExcelファイルにEmbeddedされた画像を抽出し、
ファイル名を編集してダウンロードできるブラウザツール。

## 🔗 使ってみる
https://hachimendaioh.github.io/excel-image-extractor/

## ✅ 機能
- .xls / .xlsx / .xlsm に対応
- PNG・JPEG・GIF を自動検出・抽出
- ファイル名をその場で編集
- プレフィックス（接頭辞）を全画像に一括付与
- 個別ダウンロード / ZIP一括ダウンロード

## 🔒 プライバシー
すべての処理はブラウザ内で完結。
ファイルは外部サーバーに送信されません。

## 🖥 動作環境
Chrome / Edge / Safari / Firefox（最新版）
インストール不要・サーバー不要
